## Mainstream Languages

### **main.py** - Python (1991)
General-purpose language known for readability and extensive libraries. Created by Guido van Rossum.

### **main.py2** - Python 2 (2000)
Legacy version of Python (discontinued 2020). Introduced unified types and classes.

### **main.py3** - Python 3 (2008)
Modern Python version with improved Unicode support and consistent syntax.

### **gui.pyw** - Python Windows (GUI) (1991)
Windows-specific Python files that run without console window, used for GUI applications.

### **index.js** - JavaScript (1995)
The language of the web, created by Brendan Eich in 10 days at Netscape.

### **index.ts** - TypeScript (2012)
Microsoft's statically typed superset of JavaScript with compile-time type checking.

### **main.deno.ts** - Deno (2020)
Modern TypeScript/JavaScript runtime by Ryan Dahl (Node.js creator) with security and modern APIs.

### **Main.java** - Java (1995)
Object-oriented language by James Gosling at Sun Microsystems. "Write once, run anywhere."

### **main.c** - C (1972)
Dennis Ritchie's systems programming language that defined modern computing.

### **main.cpp** - C++ (1985)
Bjarne Stroustrup's object-oriented extension of C with zero-cost abstractions.

### **main.cc** - C++ (alternative extension) (1985)
Common C++ file extension, especially in Google codebases.

### **main.cxx** - C++ (alternative extension) (1985)
Another C++ extension variant used in some build systems.

### **hello.h** - C Header (1972)
C header file for declarations and interfaces.

### **hello.hh** - C++ Header (1985)
C++ header file extension variant.

### **hello.hpp** - C++ Header (1985)
Common C++ header extension emphasizing C++ vs C.

### **hello.hxx** - C++ Header (1985)
C++ header variant matching .cxx source files.

### **Program.cs** - C# (2000)
Microsoft's object-oriented language for .NET platform by Anders Hejlsberg.

### **Program.fs** - F# (2005)
Functional-first .NET language inspired by OCaml.

### **Module1.vb** - Visual Basic .NET (2001)
Modern evolution of Visual Basic for .NET platform.

### **main.go** - Go (2009)
Google's systems language emphasizing simplicity and concurrency. Created by Rob Pike, Ken Thompson, Robert Griesemer.

### **main.rs** - Rust (2010)
Memory-safe systems language from Mozilla (Graydon Hoare). Guarantees safety without garbage collection.

### **main.swift** - Swift (2014)
Apple's modern language replacing Objective-C for iOS/macOS development.

### **hello.m** - Objective-C (1984)
Apple's primary language before Swift, adding Smalltalk-style messaging to C.

## About This Collection

This repository demonstrates the incredible diversity of programming languages created over 70+ years of computing history. From Fortran's scientific computing roots in 1957 to modern domain-specific languages for blockchain, AI, and cloud computing, each language represents a different approach to solving computational problems.

Languages evolve to address new challenges: memory safety (Rust), blockchain verification (Solidity, Cairo), parallel computing (Chapel, Bend), interactive education (Scratch, Logo), and much more. This collection showcases not just syntax differences, but fundamental philosophical differences in how we express computation.

**Key Observations:**
- **Paradigm shifts**: Imperative → Object-Oriented → Functional → Multi-paradigm
- **Domain explosion**: General purpose → Specialized (web, blockchain, GPU, audio, etc.)
- **Type system evolution**: Dynamic → Static → Dependent → Gradual
- **Safety focus**: C's freedom → Rust's guarantees → Formal verification
- **Platform adaptation**: Native → VM → Web → Edge/Embedded

Each file contains a simple "Hello, World!" program or equivalent example demonstrating the language's basic syntax and philosophy.
